# WST Assignment 3 - React useEffect Hook

A simple React application demonstrating the use of the useEffect hook with a timer-based delayed render.

## 📋 Description

This project shows how useEffect and useState work together in React. When the app loads, it waits *3 seconds* and then displays a message on the screen.

## 🚀 Features

- React 19
- useEffect hook with setTimeout
- useState for conditional rendering
- Auto cleanup with clearTimeout

## 🛠️ Installation & Setup

1. Clone the repository:
git clone https://github.com/HAM-MAD-7/WST_Assignment_3.git

2. Navigate to the project folder:
cd WST_Assignment_3/myapp2

3. Install dependencies:
npm install

4. Run the app:
npm start

## 💻 How It Works

- App loads and useEffect starts a 3-second timer
- After 3 seconds, setShowText(true) updates the state
- React re-renders and displays the message in blue
